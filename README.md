 # ⚓ WayAway  Hotel Booking Platform

**WayAway** is a full-stack hotel booking web application that enables users to discover, list, and review hotels from across India and around the world. Designed with a clean interface and powerful backend, WayAway makes it simple for travelers to explore stays and for hosts to manage their properties with ease.

##  Features

-  **User authentication** (Register / Login / Logout)
-  **Create, edit, delete hotel listings**
-  **Upload hotel images** via Cloudinary
-  **Map integration** using Mapbox
-  **Leave reviews** with ratings
-  **Access control** (Only owners can edit/delete their listings)
-  **Search listings** by location or keyword
-  **Mobile responsive** design

---

## 🛠 Tech Stack

### Frontend
- EJS (Templating Engine)
- Vanilla CSS
- Bootstrap 5 (Styling)
- Vanilla JavaScript

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose

### Dev Tools & Libraries
- Passport.js (User Authentication)
- Cloudinary + Multer (Image Uploads)
- Mapbox GL JS (Interactive Maps)
- Method-Override (Form PATCH/DELETE)
- Connect-Flash (Notifications)
- Express-Session & Cookies
